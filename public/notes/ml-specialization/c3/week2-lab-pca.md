Lecture Example

- same example as from lecture

    ```python
    X = np.array([[ 99,  -1],
           [ 98,  -1],
           [ 97,  -2],
           [101,   1],
           [102,   1],
           [103,   2]])
    ```

    ```python
    plt.plot(X[:,0], X[:,1], 'ro')
    ```

    ```python
    # Loading the PCA algorithm
    pca_2 = PCA(n_components=2)
    pca_2

    # PCA(n_components=2)
    ```

    ```python
    # Let's fit the data. We do not need to scale it, 4
    # since sklearn's implementation already handles it.
    pca_2.fit(X)

    # PCA(n_components=2)
    ```

    ```python
    pca_2.explained_variance_ratio_

    # array([0.99244289, 0.00755711])
    ```

- the coordinates on the first principal component are enough to retain 99.24% of the information
- the second component adds an additional 0.76% of the information that is not stored in the first principal component coordinates

    ```python
    X_trans_2 = pca_2.transform(X)
    X_trans_2

    '''
    array([[ 1.38340578,  0.2935787 ],
           [ 2.22189802, -0.25133484],
           [ 3.6053038 ,  0.04224385],
           [-1.38340578, -0.2935787 ],
           [-2.22189802,  0.25133484],
           [-3.6053038 , -0.04224385]])
    '''
    ```

- think of column 1 as the coordinate along the first principal component and column 2 as the coordinate along the second principal component
- could just choose the first principal component since it retains 99% of the information

    ```python
    pca_1 = PCA(n_components=1)
    pca_1

    # PCA(n_components=1)
    ```

    ```python
    pca_1.fit(X)
    pca_1.explained_variance_ratio_

    # array([0.99244289])
    ```

    ```python
    X_trans_1 = pca_1.transform(X)

    ```

- the column is just the first column of X_trans_2
- having 2 features and choosing 2 principal components, all the information will end up the same as the original


- reducing to 1 dimension instead of 2

    ```python
    X_reduced_1 = pca_1.inverse_transform(X_trans_1)
    ```

Using PCA in Exploratory Data Analysis

- load a toy dataset with 500 samples and 1000 entries

    ```python
    df = pd.read_csv("toy_dataset.csv")

    ```

- see if there is a pattern in the data
- the following function will randomly sample 100 pairwise tuples (x,y) of features,

    ```python
    def get_pairs(n = 100):
        from random import randint
        i = 0
        tuples = []
        while i < 100:
            x = df.columns[randint(0,999)]
            y = df.columns[randint(0,999)]
            while x == y or (x,y) in tuples or (y,x) in tuples:
                y = df.columns[randint(0,999)]
            tuples.append((x,y))
            i+=1
        return tuples
    ```

    ```python
    pairs = get_pairs()
    ```

- plot

    ```python
    fig, axs = plt.subplots(10,10, figsize = (35,35))
    i = 0
    for rows in axs:
        for ax in rows:
            ax.scatter(df[pairs[i][0]],df[pairs[i][1]], color = "#C00000")
            ax.set_xlabel(pairs[i][0])
            ax.set_ylabel(pairs[i][1])
            i+=1
    ```

- try PCA decomposition to compress the data into a 2D subspace

    ```python
    # Loading the PCA object
    pca = PCA(n_components = 2) # Here we choose the number of components that we will keep.
    X_pca = pca.fit_transform(df)
    df_pca = pd.DataFrame(X_pca, columns = ['principal_component_1','principal_component_2'])
    ```

    ```python
    plt.scatter(df_pca['principal_component_1'],df_pca['principal_component_2'], color = "#C00000")
    plt.xlabel('principal_component_1')
    plt.ylabel('principal_component_2')
    plt.title('PCA decomposition')
    ```

- well defined clusters can be seen

    ```python
    # pca.explained_variance_ration_ returns a list where it shows the amount of variance explained by each principal component.
    sum(pca.explained_variance_ratio_)

    # 0.14572843555106263
    ```

- and 14.6% of variance is preserved
